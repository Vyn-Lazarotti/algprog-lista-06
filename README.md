# algprog-lista-06

Lista de 7 exercícios sobre Vetores em Java.

- O primeiro algoritmo se trata de um programa que lê e mostra 5 números;
- O segundo algoritmo se trata de um programa que lê 10 números e mostra eles na ordem inversa;
- O terceiro algoritmo se trata de um programa que lê 4 notas e mostra a média;
- O quarto algoritmo se trata de um programa que lê um vetor de 10 caracteres, diz quantas consoantes foram lidas e mostra elas;
- O quinto algoritmo se trata de um programa que lê 20 números inteiros e armazenando os números pares no vetor PAR e os números IMPARES no vetor ímpar. Logo em seguida mostra os três vetores;
- O sexto algoritmo se trata de um programa que pede as quatro notas de 10 alunos, calcula e armazena num vetor a média de cada aluno, imprimindo o número de alunos com média maior ou igual a 7.0;
- O sétimo algoritmo se trata de um programa que lê um vetor de 5 números inteiros, mostra a soma, a multiplicação e os números digitados.
